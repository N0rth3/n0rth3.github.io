  • 总结
    请求的必选参数为请求页index，可选参数

    • 其它

      response

  • 对服务器请求后获取的是response对象，response对象支持下列方法
  • response.text 返回unicode数据
  • response.content 返回字节流数据
  • response.json 返回json数据
